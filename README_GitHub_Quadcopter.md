# 🚁 Autonomous Quadcopter Digital Twin using MATLAB & Simulink

## Overview

This project presents a Digital Twin of a quadcopter developed using MATLAB, Simulink, and Simscape Multibody.

The model focuses on quadcopter flight dynamics, stabilization, attitude control, motor behavior, and autonomous navigation in a simulation environment. The package delivery subsystem has been removed to create a simplified drone simulation platform suitable for research, control-system development, and digital twin applications.

---

## Features

- 3D Quadcopter Model
- Simscape Multibody Dynamics
- PID-Based Flight Controller
- Autonomous Flight Simulation
- Battery and Motor Modeling
- Wind Disturbance Analysis
- Digital Twin Environment
- MATLAB and Simulink Integration

---

## Objectives

- Simulate realistic quadcopter behavior
- Analyze flight stability and control performance
- Develop a digital twin for virtual testing
